# 基于易班的劳动二科考评系统

## 背景

基于西南石油大学劳动二课课程的背景进行开发

## 前端开发人员

- MarchRory
- aalyh

## 前端技术栈

| 分类             | 选型                                    |
| ---------------- | ----------------------------------------|
| 框架             | Vue3 全家桶                             |
| 状态管理         | Pinia                                   |
| 语言             | TypeScript                              |
| 组件库           | Vant4                                   |
| CSS 框架         | tailwindcss, 这种规模的项目用原子化方便  |
| 网络请求库       | axios                                   |
| 年度报告         | GSAP3                                   |
